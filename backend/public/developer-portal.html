Chioma Developer Portal

Build on the Chioma API with API keys or JWT authentication.

Get started

  1. Sign in with your Chioma account (use the main app or Swagger UI).
  2. Create an API key via POST /api/developer/api-keys (authenticated with JWT). Store the returned key securely; it is shown only once.
  3. Call the API with X-API-Key: your_key or Authorization: Bearer <jwt>.
